To make changes in local system and then push to GIT
1. Save locally
2. In Cmd prompt cd path/to/your/project
3. git status
4. git add .
5. git commit -m "Updated website content"
6. git push
7. wait for 30 seconds to 2 minutes
8. Open your published URL or go to Repositories->My repositories. Select Sample, click settings, go to pages, then click "Visit Site".
